Instalación de pastelones

Como todo trabajo, si no hace con la ténica adecuada no quedará óptimo. La instalación de pastelones parte como base en adecuar el terreno o superficie, elimiar restos de radier, árboles u otros. El terreno debe ser nivelado dándole las pendientes necesarias, en caso de ser una zona de transito vehicular como el estacionamiento de una casa o edificio se deberá considerar la instalación de malla en dicha zona. Siguiente a eso, hay que aplicar una base de ripio y sobre esta compactarla. Recién ahora es que el terreno estará apto para instalar los pastelones. Dicho esto, se vaciará cemento similar a la fabricación de un radier, solo que estando fresco se instalarán los pastelones, algo así como pegar cerámicas. Una vez pasado el tiempo de secado adecuado, se procederá a limpiar y lavar la superficie de restos de cemento y dejar la zona despejada para el fraguado. El proceso de fraguado es sumamente importante tanto esteticamente como para sellar aquellas junturas abiertas entre cada pastelon y evitar así la entrada de agua entre medio. Como verán, instalar pastelones tiene una serie de pasos a seguir para conseguir óptimos resultados, es por eso que debe ser realizado por expertos para no tener despues que lamentar.
